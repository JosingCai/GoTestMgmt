#### 使用方法
```
usage: smoketest.py [-h] [--host HOST] [--username USERNAME] [--password PWD]
                    [--project PROJECT] [--debug DEBUG]

Start Smoke Test in TestMgmt by project

optional arguments:
  -h, --help           show this help message and exit
  --host HOST          TestMgmt hostip or domain, default: 127.0.0.1:9033
  --username USERNAME  user for login TestMgmt, default: admin
  --password PWD       password for username, default: admin
  --project PROJECT    Input project name for smoke test
  --debug DEBUG        [Y/N] Y as Yes, N as Not, default: N
